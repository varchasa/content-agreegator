Information Aggregator

This is a web application which gathers, collects and accumulates content like news, articles, latest trends in social media, twitter feeds etc. and then displays all that information via links on one single web page.

You can click on any of the content you want to read and based on your personal interest you can read any of the category.


Contents

From this point you can see all the aspects of latest news like sports news, science news, crime news, etc. according to your prefrence and interest level.
